Features

The extensive functionality offered by Sunbird can be broadly categorized into its technical functional features. Since the aim of the project is to involve people, form communities and increase the spread of education in India, it is easy to build, manage, contribute and view resources from the portal.

Technical Features:

  • Opensource
  • Robust, scalable and extensible framework and infrastructure
  • Available on the Cloud
  • Easily configurable
  • Multilingual enabled
  • Easy for administration
  • Easy to manage
  • Data analytics

Functional Features:

  • Simple, secure user management
  • Collaboration interfaces for co-creation and partnership
  • Easy, flexible course structure definition
  • Flexible, configurable and interactive content editor
  • Configurable content creation, plugin and templates
  • Multilingual Support
  • Offline Support
  • Tool and resource library
  • Member registry
  • Reports
  • Ability to build user and organization assessment and accreditation
  • Create simple gamification features
